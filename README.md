This is my LOTR-Hobbit quiz game assistant created to play with my friends. The game is made in Polish.

1. Run the game by executing the main script in your terminal:

   python game.py

3. Personalization (Custom Backgrounds):
   To ensure compliance with copyright guidelines, the repository includes solid gray placeholders for the game backgrounds in the images/ directory and a basic cursor in the cursors/.
   If you want to fully immerse yourself in the atmosphere of Middle-earth, you can easily download your favourite fan arts and overwrite the placeholder files:
   images/background1.jpg, images/background1.jpg, cursors/normal_cursor.png, cursors/pointing_cursor.png
   The code will automatically load your custom images upon the next launch without requiring any modifications.
